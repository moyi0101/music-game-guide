# 娱悦天地 - 音乐节与游戏导览

一个现代化的 React + Vite + TypeScript + Tailwind CSS 网站，专注于音乐节和游戏资讯。

## 功能特点

- 🎵 音乐节推荐 - 发现最新最热门的音乐节
- 🎮 游戏精选 - 探索精彩的游戏作品
- 📅 活动资讯 - 获取最新活动信息
- 📱 响应式设计 - 支持各种设备

## 技术栈

- React 18
- Vite 6
- TypeScript
- Tailwind CSS 3
- Lucide React Icons

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
